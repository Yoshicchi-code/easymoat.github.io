Supported Data

Any market, any symbol — worldwide

All 20 symbol slots are fully customizable. Stocks from 30+ exchanges, plus FX, crypto, indices, commodities, and bond yields — all via Yahoo Finance.

Stocks & ETFs
Ticker + exchange suffix (e.g. AAPL, 7203.T, 0700.HK, SAP.DE)
Indices
^ + code (e.g. ^GSPC, ^N225, ^FTSE, ^GDAXI, ^HSI, ^VIX)
FX
Pair + =X (e.g. USDJPY=X, EURUSD=X, GBPJPY=X, AUDUSD=X)
Crypto
Coin-USD format (e.g. BTC-USD, ETH-USD, SOL-USD, XRP-USD)
Commodities (futures)
Code + =F (e.g. GC=F gold, CL=F crude, NG=F gas, HG=F copper)
US Treasury yields
^TNX (10Y), ^TYX (30Y), ^FVX (5Y), ^IRX (13W)

Global stock exchanges (30+)

Append the exchange suffix to the local ticker code. Search any company name on finance.yahoo.com to get the correct ticker.
